# Legal Documentation Templates

This folder contains template versions of legal documents for copying to the central Privacy-Policy repository.

## Files

- **SolarEclipse_Privacy_Policy_TEMPLATE.md** - Complete GDPR/CCPA compliant privacy policy
- **SolarEclipse_Terms_of_Use_TEMPLATE.md** - Full terms of use with safety disclaimers

## Instructions

1. **Copy these files to:** https://github.com/savargeir-byte/Privacy-Policy
2. **Rename them:**
   - `SolarEclipse_Privacy_Policy_TEMPLATE.md` → `SolarEclipse_Privacy_Policy.md`
   - `SolarEclipse_Terms_of_Use_TEMPLATE.md` → `SolarEclipse_Terms_of_Use.md`
3. **Update email addresses** if needed (currently using solareklips.com domain)

## URLs for App Store/Google Play

After copying to Privacy-Policy repo, use these URLs:

- **Privacy Policy:** `https://github.com/savargeir-byte/Privacy-Policy/blob/main/SolarEclipse_Privacy_Policy.md`
- **Terms of Use:** `https://github.com/savargeir-byte/Privacy-Policy/blob/main/SolarEclipse_Terms_of_Use.md`
